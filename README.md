# Sonoria | 听见灵魂

音乐可视化播放器，支持 20+ 种 Hydra 实时可视化风格、智能音乐特征匹配、Strudel 实时音乐生成。

## 功能

- 🎵 本地文件 / URL 音频播放
- 🎨 20+ 种可视化风格（极光、星云、火焰、水晶、故障等）
- 🤖 智能风格推荐（基于能量、频段、BPM 自动匹配）
- 🎹 Strudel 实时音乐代码生成
- 🎤 麦克风空气共鸣
- 📱 沉浸式禅模式

## 使用

直接打开 `index.html`，或部署到任意静态托管服务。

## 技术栈

- [Hydra](https://hydra.ojack.xyz/) - 可视化合成引擎
- [Strudel](https://strudel.cc/) - 实时音乐编程
- Web Audio API - 音频分析
